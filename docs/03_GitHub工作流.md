# GitHub 工作流

## 分支

- `main`：私有主线
- `exp/<name>`：实验分支
- `paper/<venue>`：投稿分支
- `revision/<venue-round>`：返修/重投分支
- `public-release`：中稿后公开分支

## Tags

- `submission-v1`
- `rebuttal-v1`
- `revision-v1`
- `camera-ready-v1`
- `public-release-v1`

## 日常流程

1. 新任务开 issue
2. 新实验或方法开 branch
3. 每个小改动 commit
4. 每次实验记录 experiment_log
5. 投稿前 tag
6. 中稿后 public-release
