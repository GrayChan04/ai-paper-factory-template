# 会议和投稿要求

每个潜在会议一个目录：

```text
venues/<VENUE_YEAR>/
  raw_official_requirements.md
  requirements.md
  checklist.md
  paper_template/
  notes.md
```

工作流：

1. 从官网复制 author guideline / CFP 到 `raw_official_requirements.md`
2. 调用 `$venue-manager-cn` 整理 `requirements.md`
3. 生成 `checklist.md`
4. 写作和投稿前检查 `paper/`
